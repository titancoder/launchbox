# launchbox

> Easily setup an express project structure and start the server in seconds.

## Installation

### Install the package globally


```sh
$ npm i -g launchbox
```
## Starting a project
```sh
$ mkdir project-name
```
```sh
$ cd project-name
```
```sh
$ launchbox -i
```
